"DNS suffixes" are used for resolving unqualified domain names (names with no dots).

In the Windows Control Panel, double-click the "Network" icon:

In the "Network" dialog, select the "Protocols" tab, select "TCP/IP Protocol", click the "Properties..." button:

In the "Microsoft TCP/IP Properties" dialog, select the "DNS" tab and add your DNS suffixes under "Domain Suffix Search Order":
